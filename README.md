# LCD-Display-library-
LCD - Liquid Crystal Display 8-Bit Mode for LPC21XX Microcontroller's.


PinOut should be fixed.
LCD ---------  MCU
-------------------------
RS ----------> P0.0
EN ----------> P0.1
RW ----------> GND //As we are only writing data.
D0 ----------> P0.2
D1 ----------> P0.3
D2 ----------> P0.4
D3 ----------> P0.5
D4 ----------> P0.6
D5 ----------> P0.7
D6 ----------> P0.8
D7 ----------> P0.9
